# Introduction

***PenneMod*** is a mod made by [Peyton (penne)](https://github.com/penne-not-pasta/). ***PenneMod*** was created to give all players a fair chance to win in Among Us, Fun **FREE** Accessories + skins, new music, and much more! This README file is a bigger ***PenneMod*** Guidebook than the itch.io page!

Lets begin.

---

## Installation:

1. Download ***PenneMod*** from the official sources.
   - [Itch.io Download](https://penne-not-pasta.itch.io/penne-mod) | [GitHub Download](https://github.com/penne-not-pasta/PenneMod/releases)
2. Unzip the `PenneMod_v_x-x-x.zip` archive.
   - x-x-x is the version number, be sure to install the latest version for updated security patches, and to ensure compatibility with the latest Among Us updates.
3. After unzipping, you will see a folder named `PenneMod_v_x-x-x`.
4. Move the `PenneMod_v_x-x-x` folder to your Among Us installation directory.
5. Launch Among Us and enjoy!
